# glaz

**glaz** — консольная Linux-утилита для мониторинга системы в реальном времени.

---

# Полная пошаговая установка и запуск

## 1. Скачать проект с GitHub

```bash
git clone https://github.com/eclipse-mq/glaz.git
```

---

## 2. Перейти в папку проекта

```bash
cd ~/glaz
```

---

## 3. Выдать право на запуск

```bash
chmod +x glaz
```

---

## 4. Запустить программу

```bash
./glaz
```

---

## 5. Установка в систему (чтобы запускать одной командой)

```bash
sudo sh install.sh
glaz
```

---

# Коротко все команды вместе

```bash
git clone https://github.com/eclipse-mq/glaz.git

```

# Возможности программы

glaz показывает в реальном времени:

- загрузку процессора (общую и по ядрам);
- температуру и частоту CPU;
- загрузку GPU (NVIDIA или AMD);
- оперативную память и подкачку;
- диски: занятое место, модель, скорость чтения/записи;
- сеть: скорость приёма и передачи;
- батарею: заряд, статус, мощность;
- процессы: RSS, CPU, THR, NI, зомби;
- uptime, load average, IP-адрес.

# Аргументы командной строки

```bash
./glaz          # сортировка по RSS (по умолчанию)
./glaz -c       # сортировка по CPU
./glaz -r       # обратная сортировка
./glaz -n 15    # показать 15 процессов
```

---

# Конфигурация

```ini
# ~/.config/resmon/config
REFRESH=3
NPROC=10
SORT=cpu
```
